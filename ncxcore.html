What is this?

NCX-Core is an application that allows you to quickly download software by NinjaCheetah and other small develpers.
Here is a list of the apps currently included:

  • CSharp Collection Vol.1
  • NCX News+
  • Lazy DSi File Downloader
  • C64 Title Loader

How do I install it?

Download the .msi file from the releases page and install it. It's that simple!
For help updating, look here.

Requirements

You will need a 64-bit version of Windows 10 with .NET 5.0 Runtime installed. (you can install that here)
