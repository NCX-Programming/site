Android: Android 8.0 or higher.

Note: WearOS is not officially supported, app will not display or function as intended.

About

RNGTool Android is our Android port of RNGTool and also allows you to generate random numbers through various customizable modes. RNGTool Android is currently in a much earlier state than RNGTool for Apple platforms, so expect missing or broken features to be improved upon in the future!

Modes currently included:

Number Mode: Generate a single random number using only a minimum and maximum number. The simplest way to generate numbers.

Dice Mode: Generate up to 6 random numbers using a variety of dice types.

Card Mode: Generate up to 7 random cards, either showing point values or faces.

Marble Mode: Generate up to 5 random letters using marbles.
