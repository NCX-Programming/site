NCX-Core-Lite

By NinjaCheetah

Platform(s): macOS, Linux

Actions Builds | View Binaries | View Source

What is it?

NCX-Core-Lite is a CLI application that allows you to quickly download software by NinjaCheetah and other small develpers.

Here is a list of the apps currently available (non-exhaustive):

  • - theVaultC
  • - Lazy DSi File Downloader
  • - C64 Title Loader

Requirements

Full dependency install instructions available on the wiki.

macOS

  • - At least macOS 10.12 (Intel, build available pre-compiled)
  • - At least macOS 11.0 (ARM, build not available pre-compiled)
  • - libcurl (install with brew install curl)
  • - GTK+ 3 (install with brew install gtk+3)

Linux

(Note that the same package names should work in pacman.)

  • - A 64-bit version of Linux
  • - libcurl (install with sudo apt install libcurl4-nss-dev)
  • - GTK+ 3 (install with sudo apt install libgtk-3-0 libgtk-3-dev)

Installing

Download the zip from the latest successful actions run, which can be accessed using the link at the top of the page, extract it, and run. That's it!

Available file formats are:

  • - A Linux ELF binary (Linux)
  • - A Unix ELF binary (macOS)
  • - A (experimental) .app binary (macOS)

Compiling

Full compiling instructions available on the wiki.

macOS

  • - Make sure you have all the dependencies installed
  • - Clone/download the source code however you prefer to
  • - Enter the directory and run make mac (for a Unix binary) or make macapp (for a macOS .app binary)

Linux

  • - Make sure you have all the dependencies installed
  • - Clone/download the source code however you prefer to
  • - Enter the directory and run make linux (makes a Linux binary)
