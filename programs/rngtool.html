RNGTool

By NinjaCheetah

Platform(s): macOS

Download Installer | View Source

What is it?

RNGTool is a macOS app used to generate random numbers. It supports multiple modes so that you can use it for multiple purposes.

Modes currently included:

Number Mode: Generate a single random number using only a minimum and maximum number. The simplest way to generate numbers.

Dice Mode: Generate up to 6 random numbers using dice with 6-20 sides. Now featuring dice icons showing your rolls when using 6-sided dice.

Card Mode: Generate up to 7 random numbers using cards. You can display the numbers, cards drawn, and the point values of them. Will output the cards as either only numbers (1-13), or including face cards (Ace, 1-10, Jack, Queen, King).

Marble Mode: Generate up to 5 random letters using marbles. Marbles with the letters on them are created automatically when you press generate.

Requirements

You will need at least macOS 11.3

Installing

To install, download and mount the DMG installer, then drag RNGTool to your Applications folder.

Compiling

To compile, use the link at the top of the page to go to the GitHub page, and then clone/download the source code however you'd like. Once you have the source, the easiest way to compile it is to open RNGTool.xcodeproj in Xcode and press the Run button.
