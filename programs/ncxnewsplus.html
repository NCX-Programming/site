NCX-News+

By NinjaCheetah

Platform(s): Windows

View Binaries | View Source

What is it?

NCX-News+ is an application that gives you the latest news about NCX software, which is all written by IanSkinner1982.

How do I install it?

Method 1 (Recommended)

Go to the NCX-News+ page in NCX-Core, and click the download button. It will be automatically installed by NCX-Core.

Method 2

You can also download the .zip archive (either from the top of the page or from GitHub) and extract it.

Requirements

You will need a 64-bit version of Windows 10 with the .NET 5.0 Runtime installed. (You can download that from here.)
